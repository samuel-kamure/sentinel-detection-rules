# Microsoft Sentinel KQL Detection Rules

A collection of KQL detection rules built and tuned for enterprise 
threat detection in Microsoft Sentinel across a multi-tenant 
environment spanning 3 countries.

## About
Built by a Threat Hunter operating across East Africa enterprise 
infrastructure. These rules are production-tested and tuned to 
reduce false positives while maintaining high-fidelity detections.

## Coverage Areas
- Identity & Authentication threats
- Endpoint anomaly detection
- Privilege escalation detection
- Lateral movement indicators
- Data exfiltration patterns
- Email-based attacks

## Environment
- Microsoft Sentinel
- Microsoft Defender for Endpoint
- 8,000+ endpoints
- Multi-tenant BPO environment
